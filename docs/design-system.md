# 1.28.306.42049 Design System v0.1

## Project Identity

**1.28.306.42049** is a visual memory archive of one solo overland journey.

- **1** solo traveller
- **28** countries
- **306** days
- **42,049** km

It is not a normal travel blog, not a product page, and not a clean commercial portfolio. It is a world folded into pages.

中文定位：**一场 solo overland journey 的视觉记忆档案——一个被折进网页里的世界。**

有些记忆会笑。  
有些记忆会发抖。  
有些记忆只是安静地留下来。

## Core Roles

- **Tuzi**: Canon owner.
- **GPT**: Sole HTML / CSS / GitHub Pages keeper.
- **Affiliates**: Visual contributors.

## Fixed Navigation

Home | Stories | Atlas | Gallery | Field Notes | About Me

Top navigation is not by continent. Continental grouping belongs inside **Atlas**.

## Page Types

- **Home**: project entrance.
- **Stories**: real journey incidents.
- **Atlas**: route map, country index, continental grouping.
- **Gallery**: curated Affiliate drawings.
- **Field Notes**: heart statements and reflections.
- **About Me**: Tuzi, Toyotaki, contact, media links.

## Emotional Categories

Comedy, Fear, Loneliness, Kindness, Wonder, Exhaustion, Border Tension, Civilisation Memory, Tenderness, Silence, Rescue, Road Fatigue, Human Warmth, Threshold, Strange Beauty.

## Visual Rule

The site does not force one drawing style. The visual style must follow the emotional truth of the memory.

## Curation Rules

Image levels:

1. Canon Image — main image of the page.
2. Alternate Interpretation — can be shown if meaningful.
3. Archive Only — kept internally, not public.

Selection is based on emotional truth, not technical beauty alone.
